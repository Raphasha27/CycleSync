# CycleSync - Premium Bicycle Management System v2.0

[![Live Demo](https://img.shields.io/badge/Demo-Live-success?style=for-the-badge)](https://raphasha27.github.io/CycleSync/)
[![GitHub](https://img.shields.io/badge/GitHub-Repository-blue?style=for-the-badge&logo=github)](https://github.com/Raphasha27/CycleSync)

A cutting-edge, feature-rich Bicycle Management System built with Vanilla JavaScript and Premium Glassmorphism UI.

![CycleSync Dashboard](screenshots/02-dashboard.png)

## ✨ Features v2.0
- **🔐 Secure Authentication**: Login system with admin credentials.
- **📊 Interactive Dashboard**: Real-time revenue charts using Canvas API with gradient effects.
- **🚲 Advanced Inventory**: Search/Filter, Add, Remove, Export to CSV.
- **👥 Customer Management**: Full customer database with rental history and analytics.
- **📜 Rental History**: Beautiful timeline view with filtering and export capabilities.
- **🔧 Maintenance Tracking**: Schedule and manage bike maintenance with priority levels.
- **📱 Mobile Responsive**: Hamburger menu and optimized layouts for all devices.
- **🎨 Premium UX**: Toast notifications, smooth animations, and glassmorphism design.
- **💾 Data Persistence**: All changes automatically saved to LocalStorage.

## 🏗️ Structure
- `index.html`: Main SPA with 5 view sections (Dashboard, Inventory, Customers, History, Maintenance).
- `css/style.css`: Premium Glassmorphism Theme with mobile-first design.
- `js/app.js`: Complete application logic (1000+ lines) with OOP architecture.

## 🚀 How to Run

### Local Development
1. Clone this repository
   ```bash
   git clone https://github.com/Raphasha27/CycleSync.git
   cd CycleSync
   ```
2. Open `index.html` in any modern web browser
3. No build step or dependencies required!

## 🎯 Key Highlights

- **Zero Dependencies**: Pure Vanilla JS - no frameworks needed
- **Lightweight**: Under 2MB total size
- **Fast**: Instant load times with no build process
- **Modern UI**: Glassmorphism design with smooth animations
- **Full CRUD**: Complete Create, Read, Update, Delete operations
- **Export Ready**: CSV export for inventory and rental data

## 🛠️ Tech Stack

- **Frontend**: Vanilla JavaScript (ES6+)
- **Styling**: CSS3 (Glassmorphism, Flexbox, Grid)
- **Charts**: HTML5 Canvas API
- **Storage**: LocalStorage API
- **Icons**: Font Awesome 6.4.0

## 📱 Responsive Design

Fully optimized for:
- 💻 Desktop (1920px+)
- 📱 Tablet (768px - 1024px)
- 📲 Mobile (320px - 767px)

## 🎨 Design System

- **Color Palette**: Neon Cyan (#00E5FF), Purple (#7000FF), Accent Blue (#2979FF)

- **Animations**: Smooth 0.3s cubic-bezier transitions
